# 集中治療科ホームページ（沖縄県立中部病院）

プロジェクト名（ブランド）: **集中治療の、今**（フッターに小さく表記。ページタイトルは「集中治療科」のまま）

## 目的

1. 一般の方向けに、当院集中治療科の理念・治療対象を紹介する
2. レジデント・フェローのリクルートのため、仕事内容を詳しく紹介する

この2つの目的を1つのページに混在させず、**役割の異なる2ページ構成**にしている。

## サイト構成

| ファイル | 対象 | 内容 |
|---|---|---|
| `index.html` | 一般向け（患者さん・ご家族） | 診療内容／診療理念／対象疾患／設備紹介／スタッフ紹介／CTA（受診・面会の問い合わせ） |
| `recruit.html` | レジデント・フェロー向け（採用） | スケジュール／数字で見るICU／教育体制／修練医の声／現場からの発信／CTA（研修・見学の問い合わせ＋紹介リンク発行）／患者さん・ご家族向けページへのクロスリンク |
| `style.css` | 共通CSS | 両ページで共有。CSSカスタムプロパティで配色を切り替え |
| `script.js` | 共通JS | 両ページで共有。スムーズスクロール、アクティブナビ、スクロールリビール、紹介リンク機能 |
| `media/echo-handling.mp4` | 動画素材 | 「現場からの発信」に掲載中の試し撮り動画（実データ） |
| `robots.txt` | 検索避け | サイト全体をクロール禁止に設定 |

両ページとも相対パスのみで組んであるため、**フォルダごとコピーすればどこにでも移設できる**（GitHub Pages・院内サーバーいずれも可）。

## デザイン

- ベースは「安心感のある病院らしいグリーン」。Noto Serif JP（見出し）＋ Noto Sans JP（本文）。
- 一般向け（`index.html`）とレジデント・フェロー向け（`recruit.html`）で **同系統だがトーンを分けている**：
  - 一般向け: グラスグリーン（`--primary: #2E9944` 系）
  - 採用向け: ティール寄りのグリーン（`--primary: #16897A` 系、`recruit.html`の`<html data-theme="recruit">`で切り替え）
- 配色トークンは `style.css` の `:root` と `:root[data-theme="recruit"]` にまとまっている。ここを触れば全体の配色を一括変更できる。

## 実装済みの主な機能

- **数字で見るICU**（`recruit.html` #stats）: 病床数・当直体制など実数が入っている項目と、まだデータがない項目（点線カード「− −」表示）を明示的に分けている。フェイクの数字は入れていない。
- **修練医の声**（`recruit.html` #residents、旧名「レジデントの声」）: 3枠とも `PLACEHOLDER` タグ付きの仮テキスト。実際のコメントに差し替え待ち。
- **現場からの発信**（`recruit.html` #dispatches）: 教育動画・記事をカードで発信する継続コンテンツ枠。現在は動画1件（「エコーの扱い方」、実データ）のみ。記事はまだ0件。
- **紹介リンク発行**（`recruit.html` CTA内）: 在籍医師が名前を入れると `?ref=名前` 付きのURLをその場で生成・コピーできる（バックエンド不要、静的サイトのまま動作）。そのリンクでアクセスすると「〇〇さんからのご紹介で…」というバナーが自動表示される（`script.js` 内、URLパラメータ読み取りのみ）。
- **noindex + robots.txt**: 実名（スタッフ）・住所を含むため、検索エンジンにインデックスされないようにしている。直リンクを知っていればアクセス可能。
